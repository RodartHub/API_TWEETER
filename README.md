# API_TWEETER
This is a little API created by FastAPI where try to use all concepts learned in the curse 'FAST API' by Facundo of Platzi.

Recently add some features:

- **USERS**:
  - Show a user.
  - Show all users
  - Delete a user.
  - Update a user.
  
- **TWEETS**:
  - Show a tweet.
  - Show all tweet
  - Delete a tweet
  - Update a tweet.
